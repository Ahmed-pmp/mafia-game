# 🎮 Mafia Game

A real-time multiplayer Mafia (Social Deduction) game built with React + Firebase. Designed for local in-person play where everyone uses their own phone.

## ✨ Features

- 🎯 9–15 players local multiplayer
- 🌐 Bilingual (Arabic + English with RTL support)
- 🎨 Dark mystery theme with atmospheric visuals
- 🔐 Private rooms with 4-digit code + password
- 👥 All roles: Mafia, Doctor, Old Man (Detective), Citizen
- 💬 Mafia private chat with "point at player" feature
- 👑 Host controls
- 📊 Vote breakdown showing who voted for whom
- 📈 End-game stats
- ⚡ Real-time sync via Firebase Realtime Database

## Tech Stack

- React 18 with Vite
- Firebase Realtime Database
- Tailwind CSS
- React Router
- Lucide React icons

## Deploy

Push to GitHub, then connect to Vercel for instant deployment.
